# Belgium

All 3 regions have Open Data addresses available.
Each of these datasets are under different licenses.
BeSt (Belgian Streets and Addresses project) collects these datasets performs some processing.

The BeSt data from all 3 regions is available under Creative Commons 4.0 Attribution.

* https://opendata.bosa.be/index.nl.html
* https://github.com/alltheplaces/alltheplaces/issues/14576#issuecomment-3547141250
