# Norway

Address data for Norway is provided by Kartverket (Norwegian Mapping Authority) through the Matrikkelen (The Cadastre).

## Data Sources

Dataset documentation: https://kartkatalog.geonorge.no/metadata/matrikkelen-adresse/f7df7a18-b30f-4745-bd64-d0863812350c

- **CSV download**: https://nedlasting.geonorge.no/geonorge/Basisdata/MatrikkelenAdresse/CSV/
